# UX Batch Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Three independent UX fixes in one PR — collapse the targets picker by default (with a selected-count badge), kill the leading blank line in rendered markdown, and surface AI checklist generation (seasonal + freeform) from the checklists pages while retiring the orphan `/suggest` route.

**Architecture:** Each task is a self-contained commit that compiles on its own (no cross-cutting type change), so per-task commits pass the repo-wide pre-commit typecheck hook. Task 3 consolidates duplicated AI-suggest logic into a `useChecklistSuggestion` hook + a shared `ChecklistPromptDialog`, reused by the dashboard card, the checklists list page, and the new-checklist page.

**Tech Stack:** Next.js (server + client components), React, Vitest + Testing Library (`@vitest-environment jsdom`), Base UI (`dropdown-menu`, `dialog`), Tailwind, `react-markdown`.

### Sub-task 3a — Shared hook

- [ ] **Step 1: Create `components/checklists/useChecklistSuggestion.ts`**

```ts
'use client';
import { useState, useTransition } from 'react';
import { toast } from 'sonner';
import type { ProposedChecklistItem } from '@/lib/ai/schemas';
import { seasonForDate } from '@/lib/ai/prompts';
import { proposeChecklist } from '@/lib/ai/suggest/checklist';

export type ChecklistPreview = {
  logId: string;
  name: string;
  description?: string;
  items: ProposedChecklistItem[];
};

export function useChecklistSuggestion() {
  const [pending, startTransition] = useTransition();
  const [preview, setPreview] = useState<ChecklistPreview | null>(null);
  const season = seasonForDate(new Date());

  function run(args: Parameters<typeof proposeChecklist>[0]) {
    startTransition(async () => {
      const r = await proposeChecklist(args);
      if (!r.ok) {
        toast.error(r.formError ?? 'Failed to generate checklist');
        return;
      }
      setPreview({
        logId: r.data.logId,
        name: r.data.name,
        description: r.data.description,
        items: r.data.items,
      });
    });
  }

  return {
    pending,
    preview,
    season,
    generateSeasonal: () => run({ mode: 'seasonal', season }),
    generateFreeform: (freeFormPrompt: string) => run({ mode: 'freeform', freeFormPrompt }),
    reset: () => setPreview(null),
  };
}
```
